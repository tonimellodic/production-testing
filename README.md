# production-testing
Production testing using puppeteer and jest

## Getting Started

### Prerequisites

- Node.js
- Yarn

### Installation

To install project dependencies:

```bash
yarn
```

## Development

### Running the tests

To run all the tests:

```bash
yarn test
```

To run smoke tests only:

```bash
yarn smoke
```

To run metrics only:

```bash
yarn metrics
```

#### Unit tests

```bash
yarn unit-test
```
