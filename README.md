# synthetic-monitoring
Obtaining synthetic metrics using puppeteer and jest

## Getting Started

### Prerequisites

- Node.js
- Yarn

### Installation

To install project dependencies:

```bash
yarn
```

## Development

### Running the tests

To run the tests:

```bash
yarn start
```

`jest` can be used to run a single test:

```bash
yarn jest <test_name>
```

#### Unit tests

```bash
yarn unit-test
```
